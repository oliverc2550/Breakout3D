Read Me for Breakout 3D

Breakout 3D can be played on the Google Chrome, Mozilla Firefox or Microsoft Edge. However the game runs best on Google Chrome
Internet Explorer is not support due to the use of the Class keyword and Async functions
Opera and Safari have not been tested
You can launch the game by double clicking/right clicking and open on the HTMLPage file in the folder, however if you run the game like this the 3D will not display due to a CORS error
The game is still full playable, you will just be missing the 3D Text
In order to get around this you can run the game on a local web server, or for ease of use launch the game through the VS Code debugger
If the browser window is resized after launching the game press f5 to refresh the window and resize the game, be aware any progress will be reset
